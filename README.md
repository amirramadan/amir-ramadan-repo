# AI Project Pattern Navigator

This is a local browser-based advisor for helping AI enthusiasts identify which
PMI Seven AI Project Pattern best fits a difficult workplace issue.

## Open the tool

Open `index.html` in a browser. No server, account, or API key is required.

## How it works

1. Describe the highest challenging workplace issue.
2. Select what AI should help with.
3. Select what data is available.
4. Set the desired autonomy level.
5. Review the recommended pattern, confidence, project brief, readiness moves,
   and comparative scores.
6. Use **Download report** to save a self-contained HTML report for stakeholder
   conversations.

The pattern definitions, data considerations, stakeholder perspectives, and
responsible-use guidance are based on the attached PMI guide,
`Seven-Patterns-of-AI.pdf`. The downloadable report structure is based on
`Applying-the-Seven-Patterns.pdf`.
